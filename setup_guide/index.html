1

전원 켜기

USB-C 어댑터(5V/2A 이상)를 본체 후면 포트에 연결합니다. 약 2초 후 디스플레이가 켜지고 부팅 화면이 나타납니다.

처음 켜는 디바이스라면

WiFi 정보가 저장되어 있지 않으므로 자동으로 설정 모드(AP 모드)에 진입합니다. 디스플레이 상단에 SmartDiffuser_Setup 글자가 보이면 정상입니다.

2

설정용 WiFi에 연결

스마트폰 WiFi 목록에서 SmartDiffuser_Setup을 선택해 연결하세요. 비밀번호는 없습니다.

  • iOS는 자동으로 설정 페이지가 열립니다.
  • Android는 알림창에 "이 네트워크에 로그인"을 누르거나, 브라우저에서 192.168.4.1을 직접 여세요.
3

집 WiFi 등록

설정 페이지가 열리면 Configure WiFi를 누르고, 사용 중인 WiFi를 선택해 비밀번호를 입력합니다.

Save를 누르면 디바이스가 자동으로 재부팅됩니다. 이 과정에서 스마트폰 WiFi는 끊길 수 있어요. 정상입니다.

주의

2.4GHz 대역만 지원합니다. 5GHz 전용 공유기를 쓰고 있다면 듀얼 밴드 모드로 바꾸거나, 폰의 핫스팟을 임시로 사용하세요.

4

메인 화면 확인

약 10초 안에 디스플레이가 메인 화면으로 전환되며, 다음 정보가 표시되면 연결 성공입니다.

날씨맑음
온도 / 습도22.4°C / 48%
잔량 1~4100%

연결이 안 됐다면

디스플레이 우측의 RE 버튼을 눌러 WiFi 정보를 초기화한 뒤, 2단계부터 다시 진행하세요.

5

영점(Tare) 잡기

잔량을 정확히 측정하려면 빈 용기 상태의 무게를 0으로 맞추는 작업이 필요합니다.

  1. 4개 향 용기를 비운 상태로 각 슬롯 정중앙에 올립니다.
  2. 디스플레이 메뉴에서 설정(또는 M3 / Y2)을 누릅니다.
  3. 잠시 기다리면 4채널 모두 0%로 초기화되고, NVS에 자동 저장됩니다.

한 번만 하면 됩니다

영점 값은 디바이스에 영구 저장됩니다. 용기 모델을 바꾸지 않는 한 다시 할 필요는 없습니다.

6

향 충전하기

각 슬롯에 디퓨저 액을 채웁니다. 기본 매핑은 다음과 같습니다.

1맑음시트러스 계열 추천
2흐림우디 계열 추천
3비아쿠아 계열 추천
4눈머스크 계열 추천

채우는 동안 디스플레이의 잔량 게이지(j1~j4)가 실시간으로 차오르는 것을 확인할 수 있습니다.

7

모드 선택해서 사용 시작

준비가 끝났습니다. 디스플레이에서 원하는 모드를 선택하세요.

  • 수동(Manual) — 4개 버튼으로 직접 분사
  • 날씨(Weather) — 현재 지역 날씨에 맞춰 자동 분사
  • 앰비언트(Ambient) — 주변 소음 분석 기반 향 추천
  • 음성(Voice) — 2초 녹음으로 음성 명령 실행

설정 완료

이제 모든 기능을 사용할 수 있습니다. 사용 중 궁금한 점은 아래 문제 해결을 참고하세요.

문제 해결

설정 WiFi(SmartDiffuser_Setup)가 보이지 않아요

이미 한 번 설정된 디바이스일 수 있습니다. 디스플레이의 RE 버튼을 길게 눌러 WiFi 정보를 초기화한 뒤 재부팅하세요. 부팅 후 다시 AP 모드로 진입합니다.

설정 페이지(192.168.4.1)가 안 열려요

모바일 데이터를 잠시 끄고 시도해 보세요. 일부 단말은 인터넷이 안 되는 WiFi에 자동으로 모바일 데이터를 우선시합니다. 그래도 안 되면 OF 버튼을 눌러 오프라인 모드로 진입한 뒤, 시리얼이나 앱을 통해 설정할 수 있습니다.

영점을 잡았는데 잔량이 음수로 나와요

용기 위치가 슬롯 중앙에서 벗어났거나, 영점 직후 용기를 살짝 들었다 놓은 경우입니다. 5단계를 다시 진행하세요.

분사 버튼을 눌러도 펌프가 동작하지 않아요

해당 채널 잔량이 15% 미만이면 보호 로직이 작동해 분사가 차단됩니다. 향을 보충해 주세요. 보충해도 동작하지 않으면 SETTING 페이지의 자가 진단으로 모터/배관을 점검할 수 있습니다.

날씨 모드가 항상 같은 향만 분사돼요

지역 정보가 비어 있거나 잘못 등록된 경우입니다. SETTING 페이지에서 지역명을 다시 입력하거나, 모바일 앱에서 위치 권한을 허용한 뒤 재동기화하세요.

디스플레이가 갑자기 꺼져요

일정 시간 입력이 없으면 자동으로 SLEEP 모드로 진입해 디스플레이를 끕니다. 화면이나 본체를 한 번 터치하면 다시 켜집니다.

알아두면 좋은 팁

모바일 앱 사용

Android 앱을 설치하면 외출 중에도 모드 전환과 잔량 확인이 가능합니다.

OTA 업데이트

새 펌웨어가 배포되면 케이블 연결 없이 WiFi로 자동 업데이트됩니다. 진행률은 디스플레이에 표시됩니다.

블렌드 모드

BM 버튼으로 활성화한 뒤 향 두 개를 순서대로 누르면, 두 향이 동시에 분사됩니다.

음성 명령 예시

"맑은 날 같은 향" / "잔잔한 향 부탁해" 같은 자연어가 인식됩니다.
